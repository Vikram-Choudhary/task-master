# Basic Overview
This Task Managing System is in PHP, CSS, JavaScript, and bootstrap. Talking about the features of this system, it contains both the admin section and the user section. The admin, can log in to assign tasks and manage the task status. The user, can log in to update task status and can also request for deadline extension.
